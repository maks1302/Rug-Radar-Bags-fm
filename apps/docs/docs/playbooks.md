## Beginner Playbook

1. Run `analyze_token`.
2. If risk is High/Extreme, skip.
3. If Medium, run `scan_risk` and review red flags.
4. Decide only after a second check.

## Active Trader Playbook

1. Run `scan_risk` first (speed).
2. Run `analyze_token` if still interested.
3. Compare with one alternative using `compare_tokens`.
4. Set watch rules and monitor deltas.

## Community Moderator Playbook

1. Run `analyze_token` for shared context.
2. Post short summary with top red flags.
3. Re-check every major market move.
4. Update community when risk materially changes.
