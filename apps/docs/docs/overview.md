title: What Rug Radar Does
---

Rug Radar is an MCP toolset for quick Solana due diligence across Claude and other MCP-compatible agents.

It helps you answer questions like:
- Is this token risky?
- Is this wallet smart money, bot-like, or random?
- Which of these two tokens is healthier?
- What changed since my last check?

## Who It Is For

- Retail traders who want fast risk context
- Researchers who need structured snapshots
- Communities that want shareable token checks

## What You Get

- Plain-English risk outputs
- Structured tool data under the hood
- Repeatable checks with watch + change tracking

Rug Radar is analysis software, not financial advice. Always DYOR.
