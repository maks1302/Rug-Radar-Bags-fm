## Top10 Holder Concentration

Percent of supply held by top 10 wallets.

## Largest Holder

Percent of supply held by the single largest visible wallet.

## Liquidity Depth

How much capital supports trading in pools.

## FDV

Fully diluted valuation. A market-cap style estimate that assumes all supply is circulating.

## Mint Authority

If active, supply can potentially be increased.

## Freeze Authority

If active, transfers can potentially be restricted.

## Honeypot Signal

Flag indicating possible buy-only/sell-restricted behavior.

## LP Unlocked

Liquidity appears unlockable or not provably burned/locked.

## Confidence

How complete and available source data was for that run.

## Baseline Snapshot

The saved previous state of a token used by `get_token_changes` for comparison.

## Watchlist

A stored monitoring rule set for a token and user.

## Dedupe Window

The time bucket used to avoid inserting the same alert repeatedly. In this project it is enforced through Redis with a default TTL of 600 seconds.
