title: When To Trust Vs Wait
---

## Trust The Output More When

- Confidence is high and no major source outages are reported.
- Multiple signals point in the same direction.
- Liquidity and holder structure are stable across checks.

## Wait For More Data When

- Confidence is low.
- A key source is unavailable.
- Token is very new and volatility is extreme.
- Metrics changed sharply within minutes.

## Practical Rule

If signal quality is mixed, wait one more check cycle.
Re-run after a short interval rather than forcing a decision.
