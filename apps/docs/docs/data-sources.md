title: Data Sources Behind Results
---

Rug Radar combines multiple sources so one API does not define the whole verdict.

## Sources

- DEX Screener: price, liquidity, volume, market context
- RugCheck: contract safety and honeypot-style flags
- Helius: holder concentration and wallet activity analysis
- Bags: ecosystem fee-share and participation context

## Why Multi-Source Matters

Different APIs capture different risk surfaces.
Combining them improves signal quality versus one-source checks.
