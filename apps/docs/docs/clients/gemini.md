title: Using Rug Radar In Gemini
---

If your Gemini setup supports MCP-compatible tool connectors, the default path is to connect the hosted Rug Radar endpoint and use it directly.

## Generic Steps

1. Add remote MCP server URL: `https://mcp.rugrdr.xyz`
2. Confirm available Rug Radar tools are detected.
3. Run prompt flow:
   - `Analyze <token>`
   - `Scan red flags for <token>`
   - `Compare <A> vs <B>`

## Note

Client UI and connector labels can change over time. Use the platform's current MCP/tool connector settings.
