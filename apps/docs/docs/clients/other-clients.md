Rug Radar is a standard MCP server. Any client with remote MCP support can integrate it.

## Connection Checklist

- MCP endpoint URL configured correctly (`https://mcp.rugrdr.xyz`)
- Transport set to Streamable HTTP
- Required headers/auth configured if your deployment uses auth
- Tool list visible after connection

## Universal Test Prompts

- `Analyze BONK`
- `Scan red flags for <token address>`
- `Analyze wallet <wallet address>`
- `What changed since last check for <token>`

If tools are not visible, verify endpoint URL and client connector settings first.
